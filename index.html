NLP Classification


In this project I used a dataset of online newsgroup posts to train a multiclass text classification model to accuractely classify these postings into one of 20 groups. The dataset comprises around 18000 newsgroups posts on 20 topics.

Skills demonstrated: Python, EDA, data processing, pandas, numpy, lambda functions NLP, NLTK, stop words, stemming, lemmatisation, part of speech tags, scikit-learn, machine learning, neural networks, keras, tensorflow, pipelines, GridSearchCV,confusion matrix, ROC AUC, accuracy metrics, logistic regression, decision tree, random forest, naive bayes, support vector machine, k nearest neighours, stochastic gradient descent, Word2Vec, GloVe vectors, data visualisation, model analysis

Tanzania Water Project


Using a dataset containing all the existing water pumps in the Tanzania, I used classification algorithms to evaluate the features that determine whether a water pump works or not. Associated with each water pump is a series of features such as: Date of installation / Water quality / Region / Installing Organisation / Source type / Nearest water bassin Length of operation. This will allow analysis of which features drive the usability of a water pump, enabling the Tanzanian government to select and evaluate more carefully third parties who install water pumps (World Bank, Anglican Church, Charities).

Skills demonstrated: python, EDA, data wrangling, data cleaning, pandas, numpy, one hot encoding, feature selection, feature creation, classification, GridSearchCV, decision tree, threshold selection, hyperparameter tuning, ensemble methods, data visualisation, geospatial data, matplotlib, seaborn

Obesity in America


The aim of this project was to understand what the drivers of obesity are in the US. The dataset analysed was from a 2019 US County Health Ranking census. An array of regression techniques were used to explore how obesity related to other physical and mental health indicators in the dataset.

Skills demonstrated: python, pandas, numpy, regression, polynomial regression, lasso regression, ridge regression, elasticnet, KFold validation, hyperparameter tuning, statsmodels, data visualisation

Movie Database Project


Using statistical methods to analyse data gathered from the Movie Database API to find out what is the most popular genre of movie and when is the best time to release a movie?

Skills demonstrated: python, pandas, API, numpy, statistical analysis, parametric and non-parametric distributions, null hypothesis, hypothesis testing, t-test, ANOVA, dunn test, tukey test, kruskal wallis test, mann whitney u test, qqplots, data visualisation, p-values, statistical significance, statistical inference

Loan Data Project/a>


In this project I used data from Lending Club to build a model to predict whether loans would be charged off. I also used the data to compare difference techniques for dealing with imbalanced datasets, as well as using Boruta feature selection to reduce the number of features from almost 1600 down to 18, while keeping recall score > 98%.

Skills demonstrated: python, pandas, numpy, imblanaced datasets, data cleaning, oversampling, undersampling, SMOTE, ADASYN, logistic regression, pipelines, GridsearchCV, feature selection, Boruta, accuracy, recall, precision, f1 score
